# Spot Work Platform

A marketplace platform that connects businesses with flexible workers for short-term shifts.

## Vision

Enable fast, reliable, and flexible job matching without long hiring processes.

## Goals

- Workers can browse and apply to jobs
- Businesses can post jobs and manage applicants
- Accepted applications become shifts
- Workers can check in and check out
- Both sides can build trust through ratings

## Project Structure

### Apps

- `apps/api` — backend API
- `apps/worker-mobile` — worker-facing app
- `apps/business-dashboard` — business dashboard

### Packages

- `packages/types` — shared types
- `packages/api-client` — API integration
- `packages/utils` — shared utilities
- `packages/validation` — input validation

### Docs

- `docs/product` — PRD, user flows
- `docs/technical` — SRD, architecture
- `docs/decisions` — design decisions

### Infrastructure

- `infrastructure/database` — DB setup
- `infrastructure/deployment` — deployment configs
- `infrastructure/scripts` — automation scripts

## Core System Modules
