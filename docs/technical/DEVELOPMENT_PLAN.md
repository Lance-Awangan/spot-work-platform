# Development Plan v1

## Goal

Build the first working **\_** of the Spot Work Platform.

---

## Scope

### Worker

- Register / login
- Browse jobs
- View job details
- Apply to jobs
- Track application status
- Check in / check out
- View completed shifts

### Business

- Register / login
- Post jobs
- View applicants
- Accept / reject applicants
- Monitor attendance
- Mark shifts complete

---

## Core System Modules

1. Auth
2. Users
3. Businesses
4. Jobs
5. Applications
6. Shifts
7. Attendance
8. Ratings

---

## Milestones

### 1. Foundation

- Repo setup
- API bootstrap
- DB connection

### 2. Auth & Roles

- Worker & business login/signup
- Role-based access

### 3. Jobs

- Create job
- List jobs
- Job details

### 4. Applications

- Apply to job
- Accept/reject applicants

### 5. Shift Lifecycle

- Application → shift
- Status tracking

### 6. Attendance

- Check-in
- Check-out

### 7. Ratings

- Worker ↔ business ratings

---

## Success Criteria

- Worker can apply to job
- Business can accept worker
- Worker can check in/out
- Shift completes successfully
- Ratings are submitted
